Introduction

This the official page for Pine-MD, an acid Markdown flavor.

About

Pine-MD is intended to be an alternative, more powerful, still simple, Markdown Syntax.

Installation

Installing Pine-MD is very simple.

Linux

$ pip install pine-md

Windows

> python -m pip install pine-md

Usage

Basic Use

Pine-MD is mostly served through its command-line interface. 
$ pine --help
A pine file (extension .mp) Now, some link, or some new tab link. Last, but not least, include a picture.
This is a simple div. Now, some lists ((8, '-'), 'First, this.') ((8, '-'), 'Second, that.') ((8, '-'), 'Third, this one.') ((8, '-'), 'Last, the fourth') ((12, '+'), 'The fifth') ((12, '+'), 'The fifth again') ((8, '-'), 'Back to the seventh') Ok, the list just ended
